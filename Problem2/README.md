## 项目背景

本实验通过构建Gompertz增长模型，使用牛顿法进行参数拟合，基于1960-198年的人口数据预测1990年人口。重点理解以下内容：
- 非线性模型的最小二乘拟合

- 牛顿法在参数优化中的应用

- 初始值对数值方法的影响
